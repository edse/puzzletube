HTML5 Video Puzzle

Building a game for learning HTML5

HTML5 Learning Features in this game:

  • Canvas
  • Image manipulation
  • RequestAnimationFrame
  • Fullscreen API
  • Application Cache API
  • Audio API
  • Mouse, keyboard, and touch support

Game Concept:

  • The player has to place all pieces in the given time
  • There are infinite stages
  • Every new stage the number of pieces and the time to accomplish increases

The game runs on canvas. It loads an image and slice it in a few pieces. The RequestAnimationFrame provide power efficient and smooth animations.
