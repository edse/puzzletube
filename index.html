HTML5 Video Puzzle

I'm using this game for learning some of the "new" HTML5 features:

  • Video API
  • Canvas
  • Image manipulation
  • RequestAnimationFrame
  • Fullscreen API
  • Application Cache API
  • Audio API
  • Mouse, keyboard, and touch support

Play »

Video, Canvas, image manipulation, and RequestAnimationFrame

HTML5 Video Puzzle runs on canvas. It loads a video and slice it in a few pieces. The RequestAnimationFrame provide power efficient and smooth animations.

Audio, and fullscreen APIs

Background music and sound effects using the Audio API. Also fullscreen mode support.

Application Cache / Offline browsing support

You can keep playing even when you go offline.
